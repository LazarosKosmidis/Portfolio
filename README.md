**Interactive Portfolio**

An interactive 3D developer portfolio built with React, Three.js, and React Three Fiber.

Live Demo

👉 https://lazaros-kosmidis.vercel.app

**Features**
-Interactive origami navigation
-Smooth GSAP animations
-Three.js 3D scene
-Floating objects
-Resume preview
-GitHub integration
-LinkedIn integration
-Responsive UI
-Modern minimalist design

**Technologies**
-React
-Three.js
-React Three Fiber
-GSAP
-JavaScript
-CSS3
